发表于 2020-12-28 更新于 2020-12-29 分类于 CODING ， dp 本文字数： 998 阅读时长 ≈ 1 分钟

给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 ‘.’ 和 ‘‘ 的正则表达式匹配，’.’ 匹配任意单个字符，’‘ 匹配零个或多个前面的那一个元素。所谓匹配，是要涵盖 整个 字符串 s的，而不是部分字符串
示例 1
输入：s = “aa” p = “a”，输出：false，解释：”a” 无法匹配 “aa” 整个字符串
示例 2
输入：s = “ab” p = “.“，输出：true，解释：”.“ 表示可匹配零个或多个（’‘）任意字符（’.’）
示例 3
输入：s = “aab” p = “cab”，输出：true，解释：因为 ‘‘ 表示零个或多个，这里 ‘c’ 为 0 个, ‘a’ 被重复一次。因此可以匹配字符串 “aab”
